A small "hello" project. The reference scaffold for the drawing board, demonstrating the per-project folder layout and design language without doing anything fancy.

## What it does

The page renders, in order:
- A mono caps eyebrow showing the project slug ("project · hello").
- A display-font heading: "Hello".
- A short description sentence.
- A `Greeter` component: a text input that echoes "Hello, {name}." as the user types, with the iris focus ring when active.
